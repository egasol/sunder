# Sunder

A lightweight nameplate addon for World of Warcraft Classic Era that displays **Sunder Armor** stack counts with clear visual feedback.

## Features

- **Stack Display**: Shows the current number of Sunder Armor stacks on enemy nameplates
- **Stack Counter**: Optional numeric badge (2-5 stacks)
- **Stack Pips**: Optional visual indicator — five small squares that fill as stacks build
- **Max-Stack Indicator**: At 5 stacks, the icon gets a steady green tint
- **Customizable**: Adjust icon size, and toggle counter/pips independently

## Commands

- `/sunder` — Toggle the addon on/off
- `/sunder on` — Enable
- `/sunder off` — Disable
- `/sunder toggle` — Toggle state
- `/sunder status` — Print current state
- `/sunder options` — Open the configuration panel

## Configuration

Open `/sunder options` to customize:
- Show Stack Counter (numeric badge)
- Show Stack Pips (five-square indicators)
- Icon Size (16–32 pixels)

All settings are saved automatically.
